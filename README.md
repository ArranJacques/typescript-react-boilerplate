# React App Boilerplate

A good starting point for React apps. Can be added to or subtracted from as required.

### Stack

- Templating
    - Nunjucks
- Javascript
    - Typescript
    - React + Redux + Immutable JS
- CSS
    - Stylus
    
## Build Tasks

#### Development server

```bash
npm run server
```

#### Development build

```bash
npm run build-dev
```

#### Production build

```bash
npm run build
```
